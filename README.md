# SnakeGame

一个零依赖的经典贪吃蛇小游戏，使用原生 HTML、CSS 和 JavaScript 实现，可直接本地启动运行。

## 启动方式

### 本地运行

```bash
npm run dev
```

启动后访问：

[http://localhost:4173](http://localhost:4173)

### 运行测试

```bash
npm test
```

## 游戏操作

- 键盘方向键或 `WASD` 控制移动方向
- 点击页面上的 `上 / 下 / 左 / 右` 按钮也可以操作
- 点击 `暂停` 按钮或按空格键可以暂停 / 继续
- 点击 `再来一次` 按钮可重新开始
- 点击 `慢 / 普通 / 快` 可以切换移动速度
- 游戏结束后按 `Enter` 也可以重新开始

## 游戏规则

- 吃到食物后，蛇身长度加 1，分数加 1
- 默认速度为 `慢`
- 撞到边缘不会结束，而是从另一边穿出
- 撞到自己身体会结束游戏
- 重新开始时会保留当前速度设置

## 项目结构

```text
.
├── docs
│   └── game-logic.md
├── index.html
├── package.json
├── styles.css
├── src
│   ├── app.js
│   └── game.js
└── tests
    └── game.test.js
```

文件职责：

- `index.html`：页面结构、按钮、棋盘挂载点
- `styles.css`：页面和棋盘样式
- `src/game.js`：纯游戏逻辑，包括移动、吃食物、碰撞、速度、重开
- `src/app.js`：DOM 渲染、事件绑定、定时器驱动
- `tests/game.test.js`：核心逻辑测试
- `docs/game-logic.md`：代码逻辑和交接说明

## 手动验证清单

- 游戏初始状态为慢速，未移动前不会自动开始
- 键盘和屏幕按钮都能控制方向
- 切换 `慢 / 普通 / 快` 后，移动频率立即变化
- 撞到边缘会从对侧出现
- 吃到食物后分数和蛇长度都会增加
- 撞到自己后显示结束提示
- `再来一次` 和 `Enter` 重开都会保留当前速度
